## Getting Started

Clone this repo:

```
$ git clone https://github.com/sirshikher/zsh-om  $HOME/.oh-my-zsh/themes/om.zsh 
```

Later go to your .zshrc file and change ZSH_THEME to this new one

```
$ cd ~/.zshrc
ZSH_THEME="om"
```
